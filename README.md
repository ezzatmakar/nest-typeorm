# NESTJS with TYPEORM
## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# used packages 
$ @nest/typeorm
$ typeorm
```
